# Installation
1. Démarrer *git bash*
1. Cloner ce repo
```
git clone https://github.com/jfgagnon00/ai-helpers.git
```
1. Initialisation
```
./initialize.sh
```

# Lancer l'environment de travail
1. Démarrer *git bash*
1. Lancer l'environment virtuel (la première exécution installera les dépendances)
```
source activate.sh
```
